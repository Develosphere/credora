# "Hey Credora" Voice Agent - User Guide

## What is it?

A hands-free voice assistant that lets you talk to your AI CFO naturally, just like Siri or Alexa.

## How to Use

### 1. Start the Voice Agent

**Option A: Say the wake word**
- Just say **"Hey Credora"** out loud
- The voice agent popup will appear automatically

**Option B: Manual trigger** (coming soon)
- Press Alt+C or click a button

### 2. Talk to Credora

Once the popup appears:
1. You'll see a pulsing microphone icon
2. **Speak your question naturally**
   - "What's my revenue this month?"
   - "Show me my top products"
   - "How are my campaigns performing?"
3. The agent will automatically detect when you stop speaking
4. Your transcript will appear on screen

### 3. Hear the Response

1. Credora will process your question
2. The response will appear as text
3. Credora will **speak the response** to you
4. You can see the full conversation in the chat window

### 4. Continue or Close

- **Continue talking**: Just start speaking again
- **Close**: Press ESC or click the X button
- **Reopen**: Say "Hey Credora" again anytime

## Features

✅ **Wake word detection**: Always listening for "Hey Credora"
✅ **Automatic silence detection**: Stops listening when you finish speaking
✅ **Voice responses**: Hear answers spoken back to you
✅ **Full context**: Has access to all your business data
✅ **Conversation history**: See past messages in chat window

- ✅ **Edge** 79+
- ❌ **Firefox** (not supported)

## Troubleshooting

### "Hey Credora" not working?
- Make sure you're on the chat page
- Check browser console for errors
- Try saying it louder or clearer
- Alternative phrases: "Hi Credora", "Hey Corridor"

### Microphone not working?
- Allow microphone access when browser asks
- Check browser settings → Site permissions
- Make sure no other app is using the microphone

### Voice not speaking back?
- Check your system volume
- Make sure browser tab is not muted
- Try refreshing the page

## Privacy

- ✅ Wake word detection runs in your browser
- ✅ Voice data is processed by browser's Web Speech API
- ✅ No audio is stored or sent to external servers
- ✅ Only text transcripts are sent to your local API
